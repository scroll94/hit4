# Моя книжка с кодами

Простой сайт для GitHub Pages с кодом pygame-игры для телефона.

## Как опубликовать

1. Создай репозиторий на GitHub.
2. Загрузи туда `index.html` и `game.py`.
3. В настройках репозитория открой `Pages`.
4. Выбери ветку `main` и папку `/root`.
5. Сайт появится по ссылке GitHub Pages.
